# Simon Says

> JavaScript implementation of [Simon's game](<https://en.wikipedia.org/wiki/Simon_(game)>)

## [Demo](https://icecoveredjune.github.io/simon-game/)

## Getting started

### Installation

```shell
npm install
```

### Build

```shell
npm run build
```

### Usage

```shell
npm start
```

Then open [localhost:8080](http://localhost:8080).
